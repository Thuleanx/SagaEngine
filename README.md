# Saga Engine

Currently with these features:
- An ECS system
- Loading and rendering .obj files with Phong lighting.
- Collision System
  - AABB collision
  - triangle-ellipsoid collision
  - optimized with a BVH tree and uniform grids for dynamic entities.
- Audio System with FMOD
